la direttiva v-model

la direttiva v-model cre associazioni dati bidirezionali su elementi di input del modulo textarea e select .
seleziona automaticamente il modo corretto per aggiornare l'elemento in base al tipo di input
v-model e essenziamenlte sintassi di zucchero per aggiornare i dati sugli eventi di input utente
piu un attenzione speciale per qualche caso limite

v-model ingnorera gli attributi iniziali value , checked o selected trovati su qualsiasi elemento del modulo .

v-model trattera sempre gli elementi vue come fonte di verita

quindi dovresti dichiarare il tuo valore iniziale su lato javascript all'interno di data

v-model usa internamente propieta diverse ed emette eventi diversi per differenziare i tipi di input
  • gli elementi text e textarea usano value value come propieta e input come evento
  • caselle di controllo e pulsanti opzione utilizzano checked come propieta ed change come evento
  • i campi di select usano value come una prop e change come evento
